# `011` last two digits

## 📝 Instrucciones:

1. Completa la función `last_two_digits()` para que dado un número entero mayor que `9`, imprima sus últimos dos dígitos.

## Ejemplo de entrada:

+ 1234

## Ejemplo de salida:

+ 34
