# `012` tens digits

## 📝 Instrucciones:

1. Completa la función `tens_digit()` para que dado un número entero, retorne sus decenas.

## Ejemplo de entrada 1:

+ 1234

## Ejemplo de salida 1:

+ 3

## Ejemplo de entrada 2:

+ 179

## Ejemplo de salida 2:

+ 7
