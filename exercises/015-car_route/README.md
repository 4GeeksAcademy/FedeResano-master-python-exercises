# `015` car route

## 📝 Instructions:

1. A car can cover distance of `N` kilometers per day. How many days will it take to cover a route of length `M` kilometers? 

## Example input:

+ 700
+ 750

## Example output:

+ 2

## 💡 Hints:

+ The program gets two numbers: `N` and `M`.
